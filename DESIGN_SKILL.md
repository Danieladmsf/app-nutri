# Anthropic Frontend Aesthetics Skill

This document guides the creation of distinctive, production-grade frontend interfaces that avoid generic "AI slop" aesthetics.

## Design Thinking

Before coding, understand the context and commit to a BOLD aesthetic direction:
- **Tone**: Choose extreme directions (Brutally minimal, luxury/refined, editorial/magazine).
- **Differentiation**: Aim for UNFORGETTABLE elements.

## Guidelines

- **Typography**: 
    - AVOID: Inter, Roboto, Arial, System fonts.
    - PREFER: Distinctive display fonts + refined body fonts (e.g., Newsreader, Playfair, Lora, or exotic Google Fonts).
- **Color & Theme**:
    - Cohesive aesthetic using CSS variables.
    - Dominant colors with sharp accents.
- **Motion**:
    - High-impact orchestrated reveals.
    - Scroll-triggering and surprising hover states.
- **Spatial Composition**:
    - Asymmetry, overlap, diagonal flow.
    - Grid-breaking elements.
- **Backgrounds**:
    - Atmosphere and depth (gradients, noise, textures).
    - Dramatic shadows and decorative borders.

## Never Use
- Generic AI aesthetics (purple gradients on white).
- Predictable layout patterns.
- Cookie-cutter designs.
